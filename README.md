# 📚 Library Manager V3

A beginner-friendly Python project that demonstrates how to build a Library Management System using Python, JSON, and file handling.

## 📌 Description

Library Manager V3 allows users to manage a collection of books through a simple command-line interface. The program stores data permanently in a JSON file so that books remain available even after closing the application.

## ✨ Features

- Add New Books
- Store Author Names
- Search Books
- View Book Details
- View All Books
- Delete Books
- Automatic JSON Data Storage
- Automatic Data Loading
- Prevent Duplicate Books

## 🧠 Concepts Used

- Python Functions
- Dictionaries
- Nested Dictionaries
- JSON
- File Handling
- Loops
- Conditional Statements
- CRUD Operations
- User Input Validation
